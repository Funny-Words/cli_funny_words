# funny_words

Simple light weight programm that logs in console funny words!

## Installation

Download release from Releases(GNU/Linux/Fedora-35 tested) or compile it yourself.

```bash
git clone https://github.com/UberNihilist/funny_words.git
```

## Usage

```bash
./main -g NUMBER
```
